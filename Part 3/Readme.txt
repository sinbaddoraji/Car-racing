Run instructions


To run the server input in command prompt

Java Server or Java Server "Port"



To run the client input in command prompt

Java GameClient or Java GameClient "HostName" Port



Game Instructions


Each client controls the cars using the arrow keys.
The game will end on the event of cars colliding. To restart the game a client has to press "R" on
the keyboard.

In the event of a player closing a client window, the server and other client will be closed


Testing

To test software using local host, look into the directory named "Test"
